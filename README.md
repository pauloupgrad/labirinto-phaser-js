# Jogo Labirinto
version 2.0

## Este jogo foi criado para aprender técnicas de desenvolvimento de jogos com javascript

## Técnicas de Aprendizado

- Trabalhar com canvas
- Criação de cenas
- Criação do player
- Criação do Inimigo
- Criação de menu
- Criação de efeitos sonoros

## Dependencias

- Phaser v2.7.5
